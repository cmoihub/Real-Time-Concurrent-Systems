Setting up
-Open up eclipse on your computer, eclipse may be downloaded from 
http://www.eclipse.org/downloads/packages/eclipse-ide-java-developers/mars1
-Go to File then Import...
-In the Import menu, under general, double click existing projects into workspace
-Click on browse to find the folder containing the files
-Click on the folder, click ok in the browse window, then click Finish in the Import window
-Right click on all files except for Host.java, go to Run As then click on Java Application
-Check the console for the results
